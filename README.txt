-------------------------------------
UNITY PROJECT SETUP
-------------------------------------

The necessary files have been included to run the Unity project.

[!] You will need to have installed a version of Unity above 2017.2 [!]

1. Extract the "Unity" file from the ZIP
2. Go into the "Assets" folder within "Unity"
3. Double Click one of the files with the suffix "-Scene" and of file type "Unity scene file"
4. Unity will open the project; if asks permission to upgrade the file to the latest version, proceed.
